# Лабораторная 1 по дисциплине "Технологии программирования" 

# Задание
Разработать программу для анализа успеваемости студентов. Система должна уметь:
	- Читать данные о студентах из разных формата JSON
	- Вычислять средний балл каждого студента  
	- Находить студентов с высокими результатами (90+ баллов по 2+ предметам) и если их несколько то выводить первого

# Средства разработчки
	- Python 3.10 - основной язык
	- pytest - для тестирования
	- Git + GitHub контроль версий
	- GitHub Actions- автоматические тесты

# Тесты
Всего тестов 12 , которые проверяют:
	- Корректное чтение данных
	- Правильный расчет рейтинга
	- Поиск студентов с высокими баллами
